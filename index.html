I've been wondering, wondering about this and built a little calculator to help me work it out.

This page should calculate the odds of a TWO players NOT winning a single bracelet (see % above). Check all events that both players will play. Set an estimated number of entrants. Set the relative skill edge to the field ((1.0 is average player, 2.0 is twice the likelyhood of winning as all other entrants, etc.)

Source for this calculator is on github. There are a ton of missing features and probably errors and major problems with this, please file them.

Feel free to contact me at @shiffman. I should work on setting some reasonable default values as well as let you save/export your selections. And tweet a link your configuration!
